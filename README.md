# CompBio-AOL
Source Code AOL CompBio - Kelompok 6

Project terdiri dari 3 files:
1. train.py - > digunakan untuk melakukan training pada machine learning
2. convert.py -> mengubah format dataset dari NCBI menjadi format yang dapat dimengerti oleh machine learning
3. predict.py -> untuk melakukan pengujian dan validasi pada machine learning

Cara penggunaan:
1. train.py -> run biasa
2. convert.py -> python convert.py test/{code_dari_dataset}_series_matrix.txt
3. predict.py -> python predict.py test/{code_dari_dataset}_series_matrix.txt
